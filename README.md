# FPBot
The friendliest potato bot there ever was.

## Quickstart
This bot relies on several environment variables being present:
* DISCORD_TOKEN
* GUILD_ID

* TWITCH_REFRESH_TOKEN
* TWITCH_CLIENT_ID
* TWITCH_CLIENT_SECRET
* TWITCH_USER
* TWITCH_USER_DISCORD
* TWITCH_USER_GITHUB

Run the command `go run main.go` to run the bot.
